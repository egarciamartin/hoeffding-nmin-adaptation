\subsection{Detailed Description}
Sufficient statistics for Entropy and Gini. 

Tracks and maintains the sufficient statistics needed to calculate Entropy and Gini of discrete and continuous attributes, as well as make some queries about the probability of events in the data.

Does not free the whole structure. A later call to Example\-Group\-Stats\-Reactivate will restore the memory (but not the counts that used to be there). This is a convienient way to focus RAM usage (and learning) in one part of the instance space while keeping the book keeping around to quickly resume learning in another. This function adds an MDL penelty similar to the one Quinlan uses in C4.5.
