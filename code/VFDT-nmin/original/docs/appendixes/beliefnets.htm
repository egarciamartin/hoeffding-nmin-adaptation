Benchmark Belief Nets

These datasets were collected from the repository at: http://www.cs.huji.ac.il/labs/compbio/Repository/

These are all the BIF format ones from that site except for two: the Barley which requires specific premission for scientific use and the Diabetes one which I downloaded an incomplete version of.

The best source I could find for the BIF format is at http://www-2.cs.cmu.edu/~fgcozman/Research/InterchangeFormat/Old/xmlbif02.html. But the authors of the format seem to have moved on to some XML based thing so I'm keeping a local copy with UWML. I have extended the format in sevaral ways so as to parse the datasets from this repository.

  • Allow a string to begin with and contain _, -, %, or 0-9
  • Allow variable values to be numbers
  • Allow 0 and 1 to be probabilities (format required 0.0 and 1.0)
Some of these supposed BIF files do not comply with this version of the BIF format. UWML also doesn't deal with noisy functions, default values, or properties.
